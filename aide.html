📖 Aide RunStats

Bienvenue dans l’application RunStats ! Elle vous permet de saisir les performances de vos élèves, calculer automatiquement la vitesse et la VMA, puis générer un QR code compatible avec ScanProf.

⚡ Lancer une saisie

Pour démarrer une saisie : ➡ Accéder à la saisie des élèves

💡 Astuce : si vous avez une modification sur votre distance de course à effectuer, signalez-le à votre enseignant avant qu’il ne scanne votre QR code… Sinon il faudra attendre l’année prochaine pour battre votre record ! 😉

📱 QR Code

  • Chaque saisie génère un QR code unique contenant les données de l’élève.
  • Ce QR code est lu par l’application ScanProf pour centraliser les résultats.

💾 Export CSV

  • Vous pouvez exporter toutes les données saisies au format CSV.
  • Ce fichier peut être ouvert dans Excel, LibreOffice, Numbers, etc.

🔑 Mode Professeur

  • Un bouton « Mode Prof » permet d’accéder à des réglages avancés (code PIN : 57).
  • Depuis ce mode, vous pouvez effacer, corriger ou ajuster les distances manuellement.

🏠 Accueil

Le bouton « Accueil » vous ramène à la page principale de RunStats à tout moment.
